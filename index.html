About this project

Background

The Bureau of Infrastructure and Transport Research Economics (BITRE) has published Australian transport statistics for decades. Its road safety enforcement dataset records roadside enforcement activities (including drug testing) from 2008 onwards. Since 2023 BITRE has captured finer-grained details (location, age group, arrests, charges) that are not fully surfaced in the current Police Enforcement Dashboard.

Aim

Design and implement interactive visualisations that expose meaningful patterns in BITRE's enforcement data, tell a clear data-driven story, and make it easy for users to explore selected parts of the dataset.

Objectives

  • Allow users to explore selected aspects of the BITRE road safety enforcement data (2008–2024).
  • Create one or more interactive visualisations that surface trends and regional differences.
  • Tell a focused story about an aspect of the data using annotation and interaction.
  • Evaluate the usability of the visualisation with basic user testing and document findings.
  • Prepare the site for deployment (the assignment expects hosting on Vercel).

This project is built by the course team as part of the Data Visualisation Project for the semester.
